# E-Commerce

A **full-stack E-Commerce web application** built with **Next.js**, **React**, and **Tailwind CSS**. This project allows users to browse products, add them to a cart, and place orders seamlessly. It includes user authentication, order management, and a responsive, mobile-friendly design.

---

## Features

- **User Authentication**
  - Sign up and login functionality
- **Product Browsing**
  - Display products by categories
  - Responsive product grid
- **Cart Management**
  - Add/remove products
  - Update quantities
  - Cart total calculation with shipping
- **Checkout and Orders**
  - Checkout form with validation
  - Place orders and store order history in local storage
  - Order summary with auto-generated order IDs
- **Notifications**
  - Toast notifications for actions like login, signup, and order placement
- **Responsive Design**
  - Works seamlessly on desktop and mobile devices

---

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS, React Toastify  
- **Backend:** Next.js API Routes, MongoDB, Mongoose  
- **State Management:** React Context API
